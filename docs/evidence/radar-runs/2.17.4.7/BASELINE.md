# Radar signal extraction baseline

До реализации `2.17.4.7` контрольный RadarRun
`radar-run-industrial-ai-triage-v2-live-20260713` находил `52` результата, выбирал
для чтения `2` разных домена и создавал `2` читаемых `FoundMaterial` без принятого
шума. `benchmarkStatus=warning` был связан с покрытием поисковой кампании.

Сигналы автоматически не извлекались: `sourceSignalCount=0`. Этот прогон служит
retrieval-базой. Новый live proof не должен ухудшить число читаемых материалов,
разнообразие доменов, принятие шума и полноту трассы; поверх retrieval должен
появиться минимум один доказательный кандидат `SourceSignal`.
